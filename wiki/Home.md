# Utah Elementary Teacher Training Assistant (UTTA)

Welcome to the UTTA documentation! This AI-powered educational simulation system helps teachers develop and refine their teaching strategies through interactive practice with simulated students.

## Key Benefits
- Practice teaching in a safe, simulated environment
- Receive immediate feedback on teaching strategies
- Experiment with different approaches
- Track progress and improvement over time

## Key Features
- **Local LLM Integration:** Privacy-focused AI processing
- **Interactive Simulations:** Realistic student responses
- **Performance Analytics:** Detailed feedback and metrics
- **Customizable Scenarios:** Various teaching situations

## System Requirements
- Python 3.8 or higher
- 8GB RAM minimum
- 10GB free disk space
- CUDA-capable GPU (recommended)
